Parcours

Comment cela se passe

Le parcours reste volontairement simple : réserver un échange, qualifier le bon angle, puis décider si une note de positionnement doit être préparée.

1. Vous réservez un créneau.
Vous choisissez directement dans l’agenda un échange éditorial de 15 minutes.

2. Nous qualifions la lecture possible.
L’échange permet d’identifier l’angle le plus pertinent, le niveau d’exposition souhaité et les éventuels angles alternatifs.

3. La suite est clarifiée.
Si une contribution semble pertinente, une note de positionnement peut être transmise. Elle précise le cadre proposé, les supports envisagés et les conditions de production.

4. La position peut ensuite être soumise au comité.
Cette soumission ne vaut pas acceptation automatique : les positions restent arbitrées selon leur pertinence éditoriale et leur complémentarité avec les autres lectures du cycle.

Questions fréquentes

Ce qu’il faut savoir avant de réserver

Faut-il préparer quelque chose ?

Non. Aucun dossier n’est attendu pour ce premier échange. L’objectif est d’abord de qualifier la lecture possible, pas de vous faire préparer une intervention.

Est-ce que cet échange vaut validation ?

Non. L’échange permet uniquement de vérifier si une suite a du sens. Une éventuelle note de positionnement pourra ensuite être transmise, puis soumise au comité éditorial si l’intervenant souhaite poursuivre.

Pourquoi un accès direct à l’agenda ?

Parce que la prise de contact a déjà eu lieu. Le parcours a donc été volontairement simplifié pour permettre un échange rapide, léger et bien cadré.

Je n’ai pas été sollicité, que faire ?

Dans ce cas, utilisez plutôt le point d’entrée Signaler une contribution, qui permet de faire connaître une position ou une lecture pertinente sans passer par cette étape réservée.

1. Renseigner vos coordonnées professionnelles

Cette première étape permet de confirmer votre demande avant d’afficher les sujets d’échange.

Votre structure

Renseignez d’abord votre organisation et votre domaine d’activité.

Votre contact

Ces informations permettent de rattacher votre demande à votre prise de contact.

2. Sélectionner le cycle concerné

Les deux cycles ne recouvrent pas les mêmes chaînes de décision. Choisissez celui qui correspond à la réalité du sujet sur lequel vous souhaitez intervenir.

3. Sélectionner la ou les conversations concernées

Une conversation correspond à un ensemble de quatre émissions individuelles mises en regard pour éclairer un même sujet sous plusieurs lectures complémentaires.

4. Préciser le contexte

Une même conversation ne se lit pas de la même manière selon les territoires. Sélectionnez, pour chaque conversation retenue, le contexte qui correspond le mieux à la réalité de vos interventions.

5. Sélectionner les angles

Choisissez maintenant les angles sur lesquels vous souhaitez intervenir. Les profils pressentis sont indiqués pour mieux situer le type de lecture attendu.
